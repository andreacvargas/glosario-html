Glosario de HTML y CSS

HTML

Hyper Text Markup Language

Es la base de una página web. Su estructura se conforma por elementos conocidos como etiquetas siendo: header, nav, article, section, aside, footer.

Etiquetas

Son códigos utilizados con el fin de dar instrucciones al navegador sonre como mostrar la informacuón, es decir, dan estructura y definen el contenido. Se escriben entre signos mayor y menor que según sea etiqueta de apertura o de cierre y entre sus funciones encontramos:

  • Estructuración de texto
  • Mostrar enlaces e imagenes
  • Etiquetas de formulario
  • Para crear tablas y divisiones
  • Etiquetas de estilo

    • Atributos

    Contienen información adicional acerca del elemento. Debe tener siempre:

  • Un espacio entre el atributo y el nombre del elemento
  • El nombre del atributo seguido por un signo =
  • Comillas de apertura y de cierre, encerrando el valor del atributo

    • ¿Cómo colocar imágenes?

    Agregar una imagen en la página se logra a través del atribiuto src (source), el cual contiene la ruta del archivo de imagen. También tiene un atributo alt que contiene un texto que podría describir la imagen para aquellos que no pueden verla.

    Su código contiene los siguientes elementos:

  • < />
  • img src=""
  • alt=""

    • Encabezados

    Los elementos de encabezado permiten especificar que ciertas partes del contenido lo son, Html posee seis niveles de encabezado, se ven de la siguiente manera:

  • < h1 > < h1 / >
  • < h2 > < h2 / >
  • < h3 > < h3 / >

    • CSS

    Cascading Style Sheets

    CSS se compone de propierdades, que son identificadores que indican qué características de estilo se quieren cambiar, y valores, que indican cómo se quiere que cambien esas características. Destacan los siguientes:

  • Bordes: Líneas que encierran el contenido de un elemento. Tiene 3 valores que son width, style y color.
  • Cajas: El diseño de CSS se basa en el modelo de caja, en el que la mayoría de los elementos HTML se pueden pensar como cajas apiladas. Cada una de ellas tiene propiedades como padding, border y margin.
  • Selectores: Permiten apuntar a elementos especñificos en una página web para aplicar estilos.
  • Especificidad: Determina qué estilo se aplica cuando hay reglas conflictivas en el código.

    • Hojas de Estilo

    Archivos externos con extensión .css que se enlazan a documentos HTML para definir su presentación.

    Comentarios en CSS

    Se utilizan para explicar el código por si en el futuro se requiere editarlo. Para que sean ignorados por los navegadores deben colocarse /* y al final */
